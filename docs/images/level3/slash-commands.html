커맨드 용도 분류
/help 현재 활성화된 모든 커맨드·스킬·MCP 서버 목록 셋업·관리
/init 현재 리포지토리에 맞춘 CLAUDE.md 초기 생성 — 코드 스캔 후 추론 셋업·관리
/config 모델·테마·권한·MCP 설정 한곳에서 토글 셋업·관리
/permissions 도구별 허용/차단 정책 편집 (allow · ask · deny) 셋업·관리
/memory 적용 중인 CLAUDE.md 파일 일괄 편집 셋업·관리
/cost 현재 세션 토큰 사용량·비용 셋업·관리
/plan 현재 요청에 한해 plan 모드로 계획 수립 (구현전 영향 범위 파악) 워크플로
/clear 대화 완전 초기화 후 작업 주제 전환 (CLAUDE.md는 유지) 워크플로
/compact 대화 요약·압축 — 핵심 결정만 보존 워크플로
/context 컨텍스트 사용량 시각화 워크플로
/model 세션 모델 전환 (Opus 4.7 ↔ Sonnet 4.6 ↔ Haiku 4.5) 워크플로
/rewind 대화 및 코드 변경 되돌리기 (결과가 마음에 안들 때) 워크플로
/review 현재 변경사항 자가 코드 리뷰 워크플로
/security-review 보안 취약점 분석 (프론트 및 api 연동 후 체크) 워크플로
기억할 것 셋업·관리는 처음 한 번 — 워크플로 명령(/plan · /clear · /compact)은 매 작업마다 손에 익혀 쓴다
