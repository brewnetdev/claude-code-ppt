필드별 의미
필드 의미
session_id 현재 세션 ID, 같은 세션 내 hook들 간 상태 공유에 사용
cwd Claude Code 실행 디렉터리
hook_event_name 이벤트 이름 (PreToolUse, PostToolUse, PreCompact, SessionStart, SessionEnd 등)
tool_name 호출되려는/된 도구 이름 (Bash, Edit, Write, Read…)
tool_input 도구 인자 객체 — 예: { "command": "npm test" }
tool_response 도구 실행 결과
transcript_path 세션의 대화 기록(JSONL 파일) 절대 경로
핵심 훅은 환경변수가 아니라 stdin(표준입력)으로 JSON 페이로드를 한 번에 받는다·jq로 파싱하는 것이 표준
