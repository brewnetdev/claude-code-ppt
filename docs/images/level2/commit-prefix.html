타입 용도 예시
feat 새 기능 추가 feat: 사용자 입력 검증 함수 추가
fix 버그 수정 fix: 로그인 토큰 만료 오류 수정
refactor 기능 변경 없이 코드 개선 refactor: 결제 모듈 중복 로직 정리
db 스키마, 마이그레이션, 쿼리 변경 db: 주문 테이블 인덱스 추가
api API 엔드포인트 추가/수정 api: 사용자 조회 엔드포인트 추가
ui 프런트엔드 UI 변경 ui: 대시보드 카드 레이아웃 변경
test 테스트 코드 추가/수정 test: 검증 함수 경계 케이스 추가
docs 문서, 주석 추가/수정 docs: README 설치 절차 보강
