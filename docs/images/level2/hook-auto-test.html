PostToolUse Hook을 활용한 자동 테스트 실행 — Edit/Write(파일을 수정할 때마다) 후 관련 테스트 자동 실행 · AI 비결정성 안전망
matcher
Claude의 파일 수정
도구 호출이 끝나면 훅 발동
hook
PostToolUse 훅 트리거
수정된 파일 경로를 환경변수로 전달
test
관련 테스트 자동 실행
변경 파일과 연관된 테스트만 골라 실행
통과
회귀 없음 — 그대로 진행
실패
즉시 발견 — 깨진 변경 차단
설정 .claude/settings.json
{ "hooks": { "PostToolUse": [ { "matcher": "Edit|Write", "hooks": [ { "type": "command", "command": "npm test -- --findRelatedTests $CLAUDE_FILE_PATH --passWithNoTests", "timeout": 60 } ] } ] } }
읽는 법 필드별 의미
"PostToolUse"
도구 실행이 끝난 직후 발동하는 훅 — 편집 결과에 반응한다
"matcher": "Edit|Write"
파일을 수정하는 도구가 끝날 때마다 훅을 발동시킨다
--findRelatedTests $CLAUDE_FILE_PATH
수정된 파일과 연관된 테스트만 선별 실행 — 전체 스위트보다 빠르다
--passWithNoTests · "timeout": 60
테스트가 없는 파일은 통과 처리, 60초 안에 끝내 작업 흐름을 막지 않는다
