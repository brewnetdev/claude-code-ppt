경로 기반 자동 컨텍스트 주입
1
규칙 파일 정의
.claude/rules/*.md 상단 frontmatter에 paths: Glob 선언
▼
2
작업 파일 경로 매칭
편집·열람 파일이 paths Glob와 일치하는지 검사
Glob MATCH
▼
3
해당 규칙만 자동 로드
일치할 때만 본문이 컨텍스트에 주입 — 매칭 안 되면 로드되지 않음
▼
4
특정 파일·디렉터리에서만 활성화
해당 영역 작업 시에만 규칙이 적용되어 컨텍스트 절약
예시 — api-routes.md
.claude/rules/api-routes.md template/claude-rules/api-routes.md 
---
paths:
  - "src/api/**/*.ts"
---
# API 개발 규칙

## 응답 형식
모든 API는 다음 형식으로 응답한다:
{ "success": boolean, "data": any | null, "error": string | null }

## 에러 처리
- 비즈니스 로직 에러: 400번대
- 서버 에러: 500번대

## 유효성 검사
- 모든 입력은 zod 스키마로 검증
- 스키마 파일은 src/schemas/에 위치
