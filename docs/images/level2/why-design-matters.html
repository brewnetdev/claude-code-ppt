AI 시대의 역설
코드 생성이 쉬워질수록,
설계가 더 중요해진다
HOW AI가 잘하는 것
'어떻게' 구현할지 — 코드 생성 · 문법 · 패턴
WHAT AI가 모르는 것
'무엇을' 만들지 — 요구사항 · 맥락 · 우선순위
코드 생성이 쉬워진 만큼 유지보수·완성도를 위한 설계가 더 중요해졌다. 'What'을 정의하는 설계는 개발자의 몫이다.
구현 전 선행되어야 할 3가지
1
요구사항 분석
무엇을 만들 것인가? 핵심 기능과 비기능적 요구사항 수립
2
아키텍처 결정
어떤 기술·구조로 만들 것인가? 빌드/배포까지 영향 — 신중히 결정
3
인터페이스 정의
기능 정의 · API 규격 · 데이터 흐름 사전 정의
