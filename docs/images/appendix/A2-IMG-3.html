상황 → 명령 빠른 참조

하고 싶은 일(좌) → 써야 할 명령(우)

하고 싶은 일 명령
파일 수정 되돌리기 (add 전) git restore <file>
add 취소 git restore --staged <file>
마지막 커밋 메시지 수정 git commit --amend -m "..."
push 전 커밋 취소 git reset --soft|mixed|hard HEAD~1
push한 커밋 되돌리기 (협업) git revert <commit> && git push
개인 브랜치 정리 git reset --hard <commit> && git push --force-with-lease
날린 커밋 복구 git reflog && git reset --hard <hash>
머지 중단 (진행 중) git merge --abort
머지 취소 (push 전 / 후) git reset --hard ORIG_HEAD  /  git revert -m 1 <merge>
