디렉토리 구조 my-plugin/
my-plugin/ ├── .claude-plugin/ │ └── plugin.json # 메타데이터(필수) ├── commands/ # 슬래시 커맨드 │ └── my-command.md ├── agents/ # 서브에이전트 │ └── my-agent.md ├── skills/ # 자동 호출 컨텍스트 │ └── my-skill/SKILL.md ├── hooks/ │ └── hooks.json # 이벤트 핸들러 └── .mcp.json # MCP 서버(선택)
4가지 구성 요소 COMPONENTS
요소트리거 / 용도
Skills 설명 매칭 시 자동 로드/도메인 지식·반복 워크플로
Commands /cmd 수동 호출/작업 실행·파일 생성
Agents 메인이 위임/독립 컨텍스트 병렬 작업
Hooks 이벤트 자동/품질 강제·로깅·보안
