@path/to/file 임포트 필요할 때만 끌어오는 컨텍스트
CONCEPT
CLAUDE.md에 적힌 @docs/api.md 경로는 자동 펼침이 아니다. 모델이 그 정보가 필요하다고 판단할 때만 Read 도구로 읽어들인다.
1 경로만 등록 CLAUDE.md에 @docs/... 참조만 적어둔다
2 평소엔 미로드 본문 토큰을 차지하지 않는다 (lazy)
3 필요 시 Read 해당 주제를 만나면 그때 읽어들인다
예시 Detailed References (lazy-loaded)
CLAUDE.md
예시 CLAUDE.md
# CLAUDE.md
## Detailed References (lazy-loaded)
- API 응답 포맷·에러코드: @docs/api-conventions.md
- DB 스키마 명명: @docs/schema-conventions.md
- 인증 흐름: @docs/auth-flow.md
- 배포 절차: @docs/deploy.md
장점 CLAUDE.md 자체는 작게 유지(50–80줄), 디테일은 별도 문서로 — 평소엔 토큰을 차지하지 않다가 모델이 해당 주제를 만나면 그때 읽는다.
