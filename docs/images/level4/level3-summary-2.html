SKILL Skills
모델이 자가 호출하는 절차 묶음. description이 호출의 전부 — 언제 써야 하는지 명확히 적는다. Slash Command와 같은 메커니즘으로 통합 운영.
HOOK Hooks
stdin JSON 페이로드 (env var 아님)로 도구 인자 수신. exit 2 = 차단 + 모델에 사유 전달, exit 1 = 비차단 경고. 8개 이벤트로 정책·자동화·핸드오프 구현.
MCP MCP
.mcp.json의 "type":"http"로 원격 서버 연결 (sse는 deprecated). 팀 공용은 .mcp.json 커밋, 개인 토큰은 환경변수로 분리.
PRACTICE 실습
프로젝트에 Skill(컨벤션) + Command(코드리뷰) + Hook(가드레일) 3종을 박는다. PR 품질·복구성·자동화가 모두 한 단계 상승.
