① PreCompact
Hook — HANDOFF.md 자동 생성
  • 컨텍스트 압축 직전에 발동
  • 현재 진행 작업 · 미해결 TODO · 다음 단계 요약을 HANDOFF.md로 dump
  • 다음 세션 시작 시 자동 임포트로 컨텍스트 복원
② SessionEnd
Hook — CHANGELOG.md 누적
  • 세션 종료 시 (정상 · crash 모두) 해당 세션의 git diff 요약 + 의사결정을 CHANGELOG.md 상단에 append
  • 날짜 · 세션 ID 메타 포함
  • 팀 데일리 빌드/공유에 활용
③ changelog-update
Skill — 모델 자가 호출
description: "사용자가 '오늘 한 일 정리해줘' 요청 시 자동 호출"
  • git log + 작업한 파일 + 결정 사항을 통합 markdown 생성
  • 팀 위클리에 그대로 붙여넣기
