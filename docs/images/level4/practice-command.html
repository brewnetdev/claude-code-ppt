.claude/commands/review.md 팀 컨벤션을 한 번에 적용하는 PR 리뷰 커맨드 — 실습 4단계
1 커맨드 파일 생성
.claude/commands/review.md
프론트매터에 description 명시 — PR 변경사항 5개 항목 (보안·성능·타입·테스트·가독성) 체크.
# review.md --- description: PR 5개 항목 리뷰 --- PR #$ARGUMENTS 받기
2 커맨드 본문
5가지 검토 카테고리
보안 입력 검증·시크릿 노출·OWASP
성능 N+1 쿼리·번들 사이즈·메모리 누수
타입 any 회피·제네릭 적절성
테스트 커버리지·엣지 케이스
가독성 네이밍·함수 길이·DRY
3 호출 패턴
/review 142 → PR #142 변경사항 fetch · 5개 카테고리별 평가 출력 · 의견을 PR comment로 게시.
gh pr comment

4 팀 공유 + 자동 적용
.claude/commands/는 git commit으로 팀 전체 공유.
GitHub Actions에서 자동 PR 리뷰 봇으로 진화.
# GitHub Actions claude -p "/review $PR"
