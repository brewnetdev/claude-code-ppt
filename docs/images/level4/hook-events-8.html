이벤트 발화 시점
PreToolUse 도구 호출 직전—차단·승인 게이트로 사용
PostToolUse 도구 호출 직후—결과 검증·로깅
UserPromptSubmit 사용자 메시지 제출 직후—입력 정제·라우팅
Notification 모델이 권한·정보를 요청할 때—알림 발송
Stop 모델 응답 종료 시
SubagentStop 서브 에이전트 종료 시
PreCompact /compact 직전—압축 전 상태 dump
SessionStart 세션 시작—컨텍스트 주입 (handoff 등)
hooks 배열 settings.json의 hooks는 이벤트별 배열 — 같은 이벤트에 여러 훅을 매처별로 등록 가능. 매처 누락 시 모든 도구에 적용된다.
