dump-handoff.sh — 쓰기
PreCompact에서 결정 · 미해결 사항을 파일로 dump. 세션이 끊겨도 컨텍스트가 사라지지 않게 만드는 안전망.
inject-handoff.sh — 읽기
SessionStart(resume)에서 dump를 읽어 새 세션 컨텍스트에 다시 주입. 끊긴 흐름을 이어 붙인다.
