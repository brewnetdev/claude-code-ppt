1.4.1 / DIRECTORY STRUCTURE
~/.claude 디렉토리 구조 — 전역 vs 프로젝트
모든 설정은 ~/.claude/(전역) 또는 현재 프로젝트의 .claude/ 디렉토리에 위치한다.
전역 USER SCOPE ~/.claude/ 모든 프로젝트에 적용
├── settings.json# 전역 설정 (모델, 권한, 환경 변수 등)
├── CLAUDE.md# 전역 메모리 (모든 프로젝트에서 로드)
├── agents# 개인 서브에이전트 정의
└── commands/# 개인 커스텀 슬래시 커맨드
    └── security-review.md# → /security-review 명령어
프로젝트 SCOPE .claude/ 프로젝트 루트
├── settings.json# 프로젝트 설정 (Git 커밋 가능, 팀 공유)
├── settings.local.json# 로컬 오버라이드 (Git 제외 권장)
├── agents# 개인 서브에이전트 정의
└── commands/# 프로젝트 커스텀 슬래시 커맨드
    └── optimize.md# → /optimize 명령어
  CLAUDE.md# 프로젝트 루트 메모리 파일 (핵심)
  CLAUDE.local.md# 로컬 전용 메모리 파일 (gitignore 권장)
우선순위 설정 적용 순서
1순위 settings.local.json 프로젝트
>
2순위 settings.json 프로젝트
>
3순위 settings.local.json 전역
>
4순위 settings.json 전역 User Scope
