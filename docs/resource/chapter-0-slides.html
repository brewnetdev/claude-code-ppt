턴 · 세션 · 컨텍스트 — 정의
세 단어를 같은 뜻으로 쓰면 강의 내내 헷갈립니다. 비유와 함께 외워둡니다.
용어 정의 비유
턴 (Turn) user 메시지 1개 + Claude 응답 1개의 쌍 (도구 호출 포함) 한 번의 질의응답
세션 (Session) claude 실행에서 종료까지의 단위. 여러 턴이 누적됨. 고유 session_id 부여 한 번의 미팅
컨텍스트 (Context) 매 추론 시점에 모델이 실제로 보는 토큰 전체. 시스템 프롬프트 + 도구 정의 + CLAUDE.md + 누적된 턴 미팅룸 책상 위의 모든 자료
💡 포함 관계: 턴 ⊂ 세션 ⊂ 컨텍스트 — 턴이 모여 세션이 되고, 세션 안의 누적된 턴 + 정적 요소가 컨텍스트를 만듭니다.

휘발성 — 무엇이 사라지고 무엇이 남는가

🔥 휘발: 컨텍스트(누적된 턴 + 도구 결과)는 세션 종료 시 모두 사라집니다. 다음 세션에 자동으로 넘어가지 않습니다.

🔗 이어가기: --continue · /resume · /compact · HANDOFF 문서 — 4가지 방법으로 세션 간 연결이 가능합니다.

Project <repo>/CLAUDE.md 이 프로젝트의 모든 협업자 ✅ 권장
Local <repo>/CLAUDE.local.md 나, 이 리포에서만 ❌ gitignore
Subfolder lazy <repo>/src/auth/CLAUDE.md 해당 폴더 작업 시만 자동 로드 ✅
