INTRO
What is title-body
Free-form blocks under one heading
Title-body slides accept arbitrary block sequences — paragraphs, h3, code, callout — in the order given.
Why we need it
Most explanatory slides do not fit the bullet template; they need short narrative passages with optional supporting code.
💡
Heads up Order is preserved exactly as authored — the renderer does not reflow.
