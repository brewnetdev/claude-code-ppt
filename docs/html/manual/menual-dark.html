사전 요구사항

Claude Code는 Node.js 18 이상을 필요로 합니다. 설치 전 다음 요구사항을 충족하는지 확인하세요.

항목 최소 요구사항 권장사항 비고
Node.js v18.0.0 v20.x LTS 권장 ESM 지원 필요
npm v8.0.0 v10.x npx 사용 가능
운영체제 macOS 12+ macOS 14 / Ubuntu 22.04 Windows는 WSL2 필요
디스크 공간 500MB 2GB 이상 모델 캐시 포함
⚠️
Windows 사용자 주의 Windows 네이티브 환경은 지원하지 않습니다. WSL2(Windows Subsystem for Linux)를 먼저 설치한 후 진행해 주세요. PowerShell 또는 Command Prompt에서는 정상 작동하지 않습니다.

Node.js 버전 확인

Terminal 
$ node --version
v20.11.0

$ npm --version
10.2.4

버전이 너무 낮다면 nvm(Node Version Manager)을 사용해 최신 LTS 버전으로 업그레이드하는 것을 권장합니다.

설치

npm 전역 설치 (권장)

가장 간단한 방법입니다. 전역 설치 후 어느 디렉토리에서도 claude 명령어를 사용할 수 있습니다.

Terminal 
$ npm install -g @anthropic-ai/claude-code

added 142 packages in 8.3s
✓ Successfully installed claude-code@2.4.1

npx를 통한 설치 없이 실행

전역 설치 없이도 npx로 즉시 실행할 수 있습니다. 단, 매 실행마다 다운로드가 발생하므로 자주 사용한다면 전역 설치를 권장합니다.

Terminal 
$ npx @anthropic-ai/claude-code

인증 설정

설치 후 Anthropic 계정과 연결해야 합니다. 두 가지 인증 방식을 지원합니다.

1
Claude Code 실행

터미널에서 claude를 입력해 처음 실행합니다.

2
인증 방식 선택

브라우저 OAuth 로그인 또는 API 키 직접 입력 중 선택합니다.

3
권한 확인

처음 실행 시 파일 접근, 터미널 실행 등 권한을 요청합니다. 프로젝트 디렉토리에서 실행하는 것을 권장합니다.

4
첫 번째 프롬프트 입력

인증이 완료되면 바로 작업을 시작할 수 있습니다.

API 키 환경 변수로 설정

~/.zshrc  또는  ~/.bashrc 
# Anthropic API 키 설정
export ANTHROPIC_API_KEY="sk-ant-api03-..."

# 선택: 기본 모델 지정
export ANTHROPIC_MODEL="claude-opus-4-5"
💡
보안 팁 API 키를 .env 파일에 직접 저장하고 .gitignore에 추가하세요. 절대로 코드 저장소에 키를 커밋하지 마세요.

주요 실행 옵션

설치 후 사용할 수 있는 주요 claude 명령어 옵션입니다.

--model
string사용할 Claude 모델 지정. 기본값은 claude-opus-4-5.
--no-auto-approve
flag파일 수정 전 매번 확인 요청. 프로덕션 환경 작업 시 권장.
--context-file
path세션 시작 시 로드할 컨텍스트 파일 경로 지정.
--max-turns
number자동 실행 모드에서 최대 대화 턴 수 제한.
--output-format
enumtext | json | stream-json 형식 선택.
