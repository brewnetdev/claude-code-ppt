1.3 Remote Control — 외부 세션 모니터링·인계
실행 중인 로컬 세션을 다른 디바이스로 가져오기 — execution stays local, view follows you
  • 실행: 여전히 로컬 머신에서코드, 파일, 환경변수가 클라우드에 업로드되지 않는다
  • 뷰(view)만 클라우드를 경유해 모바일·웹에서 보임화면 동기화만. Pro/Max 플랜 필요
시나리오사용 흐름
회의 중 작업 모니터링책상에 두고 나온 Claude Code를 회의실에서 핸드폰으로 확인
퇴근 후 야간 작업 점검집에서 Claude Code 진행 상황 확인, 필요 시 명령 입력
출장 중 빌드 모니터링호텔에서 회사 머신의 빌드 진행 상황 확인
💡
Dispatch vs Remote Control 새 작업 시작 → Dispatch. 이미 실행 중인 세션 인계·모니터링 → **Remote Control**. 보안: 모바일 분실 시 claude.ai 로그아웃 필수. 민감 작업(시크릿, 보안 코드)은 로컬에서만.

1.4 Computer Use — 시각적 자동화
Claude가 화면을 보고 마우스·키보드로 일반 GUI 애플리케이션을 조작 — API 도구로 제공
  • computer_20251124 — 2025.11 출시 최신 버전zoom action 추가: 화면 일부 영역을 고해상도로 재캡처
  • 지원 모델: Opus 4.7, Opus 4.6, Opus 4.5, Sonnet 4.6 / Haiku 미지원
  • Actions: screenshot, left_click, type, key, scroll, zoom 등

적합?

# Anthropic Quickstarts 레퍼런스 구현 실행
git clone https://github.com/anthropics/claude-quickstarts.git
cd claude-quickstarts/computer-use-demo

docker run \
    -v $(pwd):/workspace \
    -p 8080:8080 \
    -e ANTHROPIC_API_KEY=$ANTHROPIC_API_KEY \
    ghcr.io/anthropics/claude-quickstarts/computer-use-demo

# 브라우저로 localhost:8080 → 가상 데스크탑 + 채팅 UI
